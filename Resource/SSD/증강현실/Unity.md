## 유니티 물리엔진
- 유니티는 물리엔진이 내장되어 컴포넌트의 형태로 제공됨
- 따라서 물리를 구현하지 않고 컴포넌트를 사용 가능하다
- 3D와 2D는 다름

### 물리엔진의 계산 비용
- 계산비용이 높으며
- 일반적인 FPS보다 빠르게 설정됨
- 고정된 시간 0.02 1초에 50번 호출한다
	- 물리엔진을 사용할때 Update가 아닌 FixedUpdate를 호출
- 물리연산이 중요하지 않으면 조정해서 최적화함

### 강체의 뜻
- 일반적인 물리에서의 강체와는 다르다
- 유니티에서 강체
	- Rigidbody component가 없다 = 정적인 물체
	- Rigi
