## 릴레이션을 조작하기 위한 연산
### 관계 해석
무엇(What)에 대한 선언적 명시

관계형 데이터베이스의 표준 질의 언어: SQL
관계 해석(튜플 관계 해석)을 기반으로 만들어짐 무엇(What)만을 기술

### 튜플 관계 해석
튜플 변수의 범위: 릴레이션
튜플 변수는 주어진 릴레이션의 어떤 튜플도 값으로 가질 수 있음

튜플 관계 해석식에서 명시해야 하는 것

각 튜플 변수 t의 범위 릴레이션  
튜플들을 특정 조합들로 선택하기 위한 조건 검색 속성들의 집합
COND(): WFF(Well Formed Formula) => 원자(Atom)들로 구성된 식
